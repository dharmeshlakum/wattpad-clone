# Wattpad Clone - MERN Stack

## Overview

**Wattpad Clone** is a web application that replicates the core functionality of the popular **Wattpad** platform, where users can write, publish, read, and interact with stories. This project is built using the **MERN Stack** (MongoDB, Express.js, React.js, and Node.js) and provides a simple, responsive user interface for both frontend and backend development.

## Features

- **User Authentication**: Sign up, log in, and manage user profiles.
- **Story Creation**: Users can write, publish, edit, and delete stories.
- **Story Reading**: Users can browse and read stories created by other authors.
- **Likes/Dislikes**: Users can like or dislike stories based on their preferences.
- **Comments**: Users can leave comments on stories to interact with writers and other readers.
- **User Profiles**: Each user has a personalized profile displaying their stories.
- **Responsive UI**: The application is responsive and works well on desktop and mobile devices.

## Tech Stack

- **Frontend**:
  - **React.js**: JavaScript library for building user interfaces.
  - **React Router**: For managing navigation between pages.
  - **Axios**: For making API calls to the backend server.
  - **Bootstrap** or **Material UI**: For responsive, pre-built UI components.

- **Backend**:
  - **Node.js**: JavaScript runtime for building the server.
  - **Express.js**: Framework for building RESTful APIs in Node.js.
  - **MongoDB**: NoSQL database for storing user data, stories, and comments.
  - **Mongoose**: ODM for MongoDB to model application data.
  - **JWT (JSON Web Token)**: For secure user authentication.

- **Development Tools**:
  - **Git**: Version control system.
  - **Postman**: For testing backend API endpoints.
  - **MongoDB Atlas**: Cloud-hosted MongoDB for production.
